# music-app
DeltaXAssignment


## Project Statement : 

 Create a Spotify like website with basic CRUD and song listings. 
 We would like to have the following entities in the application. 
#### Artists
 - Name
 - DOB
 - Bio
#### Songs
 - Name
 - Date of Release
 - Cover (image)
#### User
 - Name
 - Email


## Tech Stack used : 
### Frontend
- HTML
- CSS
- React.js

### Backend
- Node.js
-Express.js

### Database
- MongoDB
